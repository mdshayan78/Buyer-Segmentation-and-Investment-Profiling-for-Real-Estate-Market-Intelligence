# 🏠 Buyer Segmentation and Investment Profiling for Real Estate Market Intelligence

## 📌 Project Overview

This project uses Machine Learning to segment real estate buyers based on their demographics, investment behavior, loan preferences, and purchasing patterns. The objective is to help real estate companies identify different customer groups and improve marketing and investment strategies.

---

## 🎯 Problem Statement

Real estate companies often treat every buyer the same, resulting in:

- Inefficient marketing campaigns
- Poor customer targeting
- Generic property recommendations
- Missed investment opportunities

This project solves these challenges using K-Means Clustering.
