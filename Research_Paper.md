# Machine Learning Based Buyer Segmentation and Investment Profiling for Real Estate Market Intelligence

## Abstract

This project applies Machine Learning techniques to identify hidden buyer segments in the real estate market using customer demographic and transaction data. K-Means clustering is used to group buyers with similar purchasing behavior, helping organizations improve marketing strategies, customer targeting, and investment decision-making.

---

## Introduction

The real estate market consists of different types of buyers such as first-time home buyers, investors, and corporate clients. Traditional analysis treats all buyers similarly, reducing marketing efficiency. Buyer segmentation using Machine Learning helps discover hidden customer patterns and enables personalized recommendations.

---

## Problem Statement

The company lacks a data-driven understanding of:

- Buyer types
- Investment behavior
- Geographic trends
- Loan preferences

This results in inefficient marketing and poor customer targeting.

---

## Dataset

The dataset contains buyer demographic information and property transaction details, including:

- Client Type
- Country
- Region
- Acquisition Purpose
- Loan Applied
- Satisfaction Score
- Sale Price
- Floor Area

---

## Data Preprocessing

- Removed duplicate records
- Converted sale price to numeric values
- Merged client and property datasets
- Encoded categorical variables
- Standardized numerical features using StandardScaler

---

## Machine Learning Model

Algorithm Used:

- K-Means Clustering

Evaluation:

- Elbow Method
- Silhouette Score

---

## Results

The clustering model successfully identified multiple buyer segments based on demographic and purchasing characteristics.

These segments can be used for:

- Personalized marketing
- Better investment profiling
- Improved customer targeting

---

## Conclusion

Machine Learning based buyer segmentation enables real estate companies to make informed decisions and improve marketing efficiency through data-driven customer insights.
